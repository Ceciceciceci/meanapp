Rentennial (2016)

Rentennial is a cradle-to-grave application which allows anyone to be a landlord or tenant with full access and ability to search listings, save listings, make listings, apply for listings, sign leases, and pay for rent.

I worked on the front end using AngularJS, CSS3, and JS. I also designed the UI for the page. The username and password needed to get into the website to play with it is: fiona and rentennial

CRUD, Django, AngularJS, python; The github repository is currently private.
